RIVAL10 Dataset

RIch Visual Attributes with Localization

26k Images over 10 classes ♦ 18 Visual Attributes Per Image ♦ Attribute and Object Segmentations

Classes

    RIVAL10 adopts CIFAR10 classes via Imagenet.
    That is, two Imagenet synsets are combined for each RIVAL10 class, as visualized above.

Attributes


    Every image has binary annotations for 18 informative and intuitive visual attributes, meaning:

    ♦       Any two classes can be distinguished by at least one attribute (informative)
    ♦       A human classifying RIVAL10 images may appeal to these attributes (intuitive)

Download


  • From Github (recommended)

    Follow the setup instructions in this repository to download the data and other utilities.
    This repo also details how to replicate all experiments.

  • Directly From Box

    Download the dataset directly from box, or use the following code to download the data on remote server: 
    curl -L https://umd.app.box.com/shared/static/eyji8leh2lemhbxovgny9ywc9is53ibr -o salient_imagenet_dataset.zip
    
    unzip salient_imagenet_dataset.zip

    • We recommed the first approach, where you can also access easy-to-use data loaders that consolidate annotations to a single core mask per image, as well as evaluation procedures.
